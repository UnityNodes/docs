This pattern helps you find real, working Daml code quickly when you need a practical starting point rather than reading through formal specifications.

## Find working Daml code by topic

<Note>**Problem.** Forum threads contain working Daml code but general web search rarely surfaces them reliably.</Note>

**Solution.** `find_code_examples(topic)` returns forum threads with embedded code blocks matching the topic. This is faster than reading the spec when you need a working sketch to adapt.
