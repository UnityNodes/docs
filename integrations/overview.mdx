## Notes for all clients

Keep these points in mind regardless of which client you use.

**Rate limit.** CCPEDIA allows 60 calls per 15-minute window per IP address. Exceeding the limit returns a `429` response with a `Retry-After` header. If you need higher throughput, the source is Apache 2.0 and you can self-host.

**Transport details.** The endpoint speaks JSON-RPC 2.0 over streamable HTTP.

| Property | Value |
|---|---|
| URL | `https://ccpedia.xyz/mcp` |
| Method | `POST` |
| Content-Type | `application/json` |
| Accept | `application/json, text/event-stream` |
| Auth | None |

**Older stdio clients.** Some clients — including Claude Desktop and Zed — expect a stdio process rather than an HTTP endpoint. Use the [`mcp-remote`](https://www.npmjs.com/package/mcp-remote) bridge to translate between them. Each client guide below shows the exact command. You need Node 18 or newer for npx to resolve the package automatically.

```bash
npx -y mcp-remote https://ccpedia.xyz/mcp
```

**Guided setup wizard.** If you prefer a browser-based walkthrough, the wizard at [ccpedia.xyz/for-ai](https://ccpedia.xyz/for-ai) auto-detects your client and pings the server live to confirm connectivity before you switch to your IDE.
